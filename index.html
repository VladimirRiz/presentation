Introduction

  • most powerful layout system in CSS
  • 2D (columns & rows)

How it works:

  • apply CSS rules to parent element and to its children

Goal of CSS Grid Layout:

to completely change the way we design grid-based UI

Previously:

  • tables
  • floats
  • positioning
  • inline block

Flexbox - helpful, but only 1D

Grid - specifically created to solve layout problems

Basics and Browser Support

  • supported by nearly all browsers
  • the time to build with grid is now!

Getting Started

  1. define container element as a grid with display: grid
  2. set column and row sizes with grid-template-columns and grid-template-rows
  3. place child elements into the grid with grid-column and grid-row.

- the number under each browser is the earliest version of grid supported by that browser

Important Terminology

Grid Container

The element on which display: grid is applied. It’s the direct parent of all the grid items. In this example container is the grid container.

Grid Item

The children (i.e. direct descendants) of the grid container. Here the item elements are grid items, but sub-item isn’t.

Grid Line

  • dividing lines that make up the structure of the grid
  • can be vertical, horizontal, on either side of row/column
  • example: the yellow line is a column grid line

Grid Cell

  • space between two adjacent row and two adjacent column grid lines
  • single unit of grid
  • example: the grid cell between row grid lines 1 and 2, and column grid lines 2 and 3

Grid Track

The space between two adjacent grid lines.

Grid Area

  • total space surrounded by 4 grid lines
  • may be composed of any number of grid cells
  • example:the grid area between row grid lines 1 and 3, and column grid lines 1 and 3.
